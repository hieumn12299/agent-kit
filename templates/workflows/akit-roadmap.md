# /akit-roadmap

> View the Agent-Kit roadmap and current progress.

## FIRST: Read RULES

Read `.agent/RULES.md` and follow all rules.

## THEN: Present Roadmap

1. Read `docs/roadmap.md` from the project root
2. Read `.agent/config.yaml` for `communicationLanguage`
3. Present the roadmap in the user's language with:
   - Current phase and progress
   - Completed vs remaining epics
   - Next priority items
4. Ask user if they want to:
   - Focus on a specific epic
   - Update status of completed items
   - Add new items to the roadmap

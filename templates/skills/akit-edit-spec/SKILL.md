# Edit Spec Skill

Guides AI through loading, analyzing, updating, and validating changes to existing specs:
1. 📄 Identify — which spec to edit
2. 📝 Understand — what needs to change and why
3. ⚠️ Impact Analysis — check dependencies and scope changes
4. 🔧 Apply — diff-based changes with before/after
5. ✅ Validate — consistency and completeness check
6. 📝 Save — decision memory + changelog

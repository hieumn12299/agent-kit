# Bug Fix Skill

Guides AI through a structured 6-step bug-fixing process with enforcement gates:
1. 🐛 Bug Report — gather and format the issue
2. 🔄 Reproduce — confirm the bug exists
3. 🔍 Root Cause Analysis — systematic investigation
4. 🔧 Fix — smallest possible fix
5. ✅ Verify — run tests, check regressions
6. 📝 Save Learning — create bug-learning memory to prevent recurrence

Includes `investigation-checklist.csv` for systematic analysis.
